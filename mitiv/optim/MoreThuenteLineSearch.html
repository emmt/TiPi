      • iterateHook

        protected int iterateHook(double s1,
              double f1,
              double g1)
        Description copied from class: LineSearch
        Protected abstract method to iterate during a line search. This protected method is called by the LineSearch.iterate(double, double, double) method to check whether line search has converged and, otherwise, to compute the next step to try (stored as attribute stp). The provided arguments have been checked. Upon return, the caller method, LineSearch.iterate(double, double, double), takes care of safeguarding the step.
        Specified by:
        iterateHook in class LineSearch
        Parameters:
        s1 - The value of the step (same as the value returned by LineSearch.getStep()).
        f1 - The value of the function at x1 = x0 + s1*p where x0 are the variables at the start of the line search and p is the search direction.
        g1 - The directional derivative at x1, that is p'.g(x1) the inner product between the search direction and the function gradient at x1.
        Returns:
        The new status of the line search instance.
