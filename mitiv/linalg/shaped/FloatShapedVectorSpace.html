      • create

        public FloatShapedVector create(ShapedArray arr)
        Description copied from class: ShapedVectorSpace
        Create a new vector initialized with the contents of an array.

        The values of the elements of the returned vector will be copied (or shared, see below) or converted from those of the input array. Type conversion is automatically performed but the shape of the array must match that of the vectors of this space.

        Depending on the array and on the vector storages, the vector and the array may share the same data but there is no guarantees for that. To save memory, the default behavior is to share contents if possible. If you want to make sure that the two objects have independent contents, call ShapedVectorSpace.create(ShapedArray, boolean) with second argument set to true.

        In any cases, if the array data storage is not "flat" or if its elements type does not match that of the vector, the two contents will be stored independently.
