      • zeroPadding

        public static ShapedArray zeroPadding(ShapedArray inputArray,
                      Shape outputShape)
        Zero-pad a shaped array.

        Zero-padding consists in adding zeros around an array to build a larger array.

        There must be as many dimensions in the result as in the input array and all dimensions must be greater or equal the corresponding dimension in the input array. The operation is lazy: if no padding is needed (that is, if the shapes are the same), the input array is returned. Otherwise, the contents of the input array is pasted into a larger output array approximately at the geometrical center of this former array. More specifically, the number of zeros at the beginning of a given dimension is equal to: 

         (outDim/2) - (inpDim/2)
        assuming integer arithmetic and where outDim and inpDim are the respective length of the given dimension in the output and input arrays.

        Parameters:
        inputArray - - The input array.
        outputShape - - The shape of the result.
        Returns:
        A shaped array of the given shape.

      • roll

        public static ShapedArray roll(ShapedArray arr,
               int[] off)
        Roll the dimensions of a shaped array with given offsets.

        This static method rolls the contents of the input array along its dimensions. For a mono-dimensional array of length dim, this is equivalent to something like: 

         dst[j] = src[(j - off)%dim]
        where src is the input array, dst is the result, off is the offset and assuming that the modulo operator returns a result wrapped in the range [0:dim-1].

        The operation is lazy: if no rolling is needed (that is, if the shapes are the same), the input array is returned.

        Parameters:
        arr - - The input array.
        off - - The offsets to apply along each dimensions.
        Returns:
        A shaped array with the contents of the input array but rolled along the dimensions of the input array by the given offsets. Note that the result shares its contents with the input array.
