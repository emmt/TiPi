        public FiniteDifferenceOperator(DoubleShapedVectorSpace inputSpace,
                        BoundaryConditions[] bounds)
        Create an instance of a finite difference operator.

        The finite difference operator, computes the difference between each element of its argument and its preceding element for each dimensions. Thus a 1D array yields a 1D result while a n-dimensional array yields a n + 1 dimensional result with an additional leading dimension of length n to store the finite differences along each of the n dimensions.

        Parameters:
        inputSpace - - The inputs space of the operator (the output space is automatically built).
        bounds - - An array indicating the boundary conditions for each dimension. If null, normal conditions are assumed for all dimensions; otherwise, if shorter than n, the rank of inputSpace, missing values are assumed to be BoundaryConditions.NORMAL.
